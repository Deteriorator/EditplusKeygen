## 如何区分注册码的版本
注册表里
```
HKEY_CURRENT_USER\SOFTWARE\ES-Computing\EditPlus\Install
```
### 4.x版本中

License 表示 3.x 的注册码

License 4 表示 4.x 的注册码

### 5.x版本中

License 4 表示 3.x 或 4.x 的注册码

License 5 表示 5.x 的注册码

都有的话软件优先判断高版本的

`经测试 3.x 和 4.x 的注册码均可用于注册 5.x 的`
