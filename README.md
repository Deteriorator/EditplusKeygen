## 如何区分是4.x还是3.x的注册码
注册表里
```
HKEY_CURRENT_USER\SOFTWARE\ES-Computing\EditPlus\Install
```
License 表示3.x的注册码

License 4 表示4.x的注册码

都有的话软件优先判断4.x的
